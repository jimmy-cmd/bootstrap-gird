Curso CSS y HTML
Aprenderas a manejar html, css, algunas otras funciones.

Con el lenguaje HTML se crean las páginas web, pero no su diseño, mientras que con CSS (un lenguaje de hojas de estilos), se le da a esos contenidos una apariencia atractiva para el usuario.

Curso Bootstrap
Aprenderas a usar Framework.

Bootstrap, es un framework originalmente creado por Twitter, que permite crear interfaces web con CSS y JavaScript, cuya particularidad es la de adaptar la interfaz del sitio web al tamaño del dispositivo en que se visualice.

Curso Java
Html5 y sus estilos dedfinidos

Con JavaScript se puede interactuar y modificar cualquier objeto de una página HTML5 y sus estilos definidos con CSS3: textos, capas, estilos, imágenes, videos, audios, formularios, etc. HTML5 y CSS3 por si solos no pueden usarse para conseguir interactividad y animación, para ello necesita JavaScript.
